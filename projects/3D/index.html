Here are some other 3D modeling projects that didn't really need a page for their own. Most were done as fun/training projects, with no specific purpose in mind.

01 // Rogue one u-wing concept

I found some old concept art for the Star Wars Rogue one movie for the iconic U-wing, so I decided to try and recreate this concept in Blender.

02 // Moto Guzzi 1200 Custom

This project also started with some concept art I found online. All main parts were modeled conventionally with some use of kitbashing techniques for detail clusters. Modeled and rendered in Blender.

03 // Overland 450

This concept motorcycle is based around a Honda CRF450. I wanted to explore a futuristic, almost cyberpunk approach to motorcycle design, but to still work with existing hardware to keep the concept rooted in reality. The design is inspired by this custom motorcycle build. An existing model was used as the base, so this was mainly a rendering exercise.
